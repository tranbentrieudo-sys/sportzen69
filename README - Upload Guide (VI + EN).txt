🇬🇧 ENGLISH GUIDE
-----------------
1️⃣ Go to: https://app.netlify.com/
2️⃣ Sign up for a free account (Google / Email)
3️⃣ After logging in, choose “New site from drag & drop”
4️⃣ Drag and drop your SportZen69.html file (and favicon-69.png if you want the logo)
5️⃣ Wait a few seconds → Your site will go live permanently at:
   👉 https://yourname.netlify.app
6️⃣ To change the name, go to “Site settings → Domain management”.

📎 Tip to add a favicon:
   - Upload favicon-69.png in the same folder as your HTML.
   - Add this line inside <head>:
     <link rel="icon" type="image/png" href="favicon-69.png">
